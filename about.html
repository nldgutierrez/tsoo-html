    About this project

    Hi! I'm Noelle Gutierrez, currently self-studying Web Development from The Odin Project and Computer Science from OSSU, and an aspiring Software Engineer.

    The Science of Obsidian is a small website project of mine. Its content are primarily ripped off of other sources in the web (see: References) to give me something to work with for its structure.

    The main objective for this project is to practice my skills in the foundations of web development--specifically, HTML, CSS, and JavaScript.

    This website's main topic is Obsidian, a note-taking software that I discovered recently and am loving more each day. I did not immediately think of this as the topic for the website. In fact, I had an initial list of five (5) different topics and used Wheel Decide to pick one for me, which is Obsidian.

    This project is divided into three stages: HTML, HTML+CSS, and HTML+CSS+JavaScript. Pretty self-explanatory, but basically, I would like to create three versions of this website to practice building websites properly. Especially for HTML, I wanted to learn accessibility and semantic HTML right from the start so I could create structure with context and in the future, accessible websites.

    All stages of this project are available on my GitHub profile.

    Should you wish to give any comment, suggestion, or feedback on how I could further improve my skills, please do not hesitate to send me a message.

    Have a great day! :)
